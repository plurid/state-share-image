Demonstration for the State-Share Image Proposal

Open this web page in two distinct browser tabs/windows, or even two different browsers.

Change the theme of the page by clicking on the buttons below, and thus altering the state.

  • night
  • gradient
  • light 
state = {
    app: {
        theme: 'night'
    }
}

Click on the state-share-image and copy it to clipboard.

Switch to the other tab/window/browser, click on the state-share-image and paste it.

The state of the application has been succesfully encoded into the image (which could be shared with other users or saved externally for later use/referral in a State-Share Images Manager) and then succesfully converted back into a state object useful for the application, albeit minimal in this particular demonstration.
